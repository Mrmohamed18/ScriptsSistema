Este repositorio contiene los scripts de automatización desarrollados durante las 4 semanas. la gestión de sistemas mediante Bash (Ubuntu) y PowerShell ISE(Windows).

Las Semanas:

Bash (Ubuntu)

Semana 1 / Infosistema: Script para extraer información básica del hardware.

Semana 2 / Usuaris: Automatización del alta masiva de usuarios desde un archivo .txt.

Semana 3 / Backup: Script de copia de seguridad local con registro de logs y automatización mediante Cron.

Semana 4 / Monitoratge: Script de control de procesos críticos y generación de informes de estado y tambien hacemos en cron pero opcional.

PowerShell ISE (Windows)

Semana 1 / Infosistema: Obtención de datos del sistema mediante cmdlets nativos.

Semana 2 / Usuaris: Creación masiva de cuentas locales gestionando permisos de administrador y políticas de ejecución.
